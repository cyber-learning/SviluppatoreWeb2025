Selettori CSS utili per selezionare gli elementi HTML su cui
vogliamo applicare le dichiarazioni CSS:
    *   universal selector permette di selezionare tutti gli elementi HTML della pagina
    tag selector permette di selezionare tutti gli elementi HTML della pagina avente quel determinato tag
    .nomeDellaClasse class selector
    #id id selector permette di selezionare SOLO un elemento HTML
