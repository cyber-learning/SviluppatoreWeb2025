Margin collapsing: il margine verticale tra elementi siblings (fratelli) collassa 

block level element:
- di default hanno width 100%
- sono impliati in verticale

inline level element:
- hanno width strettamente necessaria a contenere il proprio contenuto
- sono impliati in orizzontale


Flexbox attori:
- 1 flex container
- uno o più figli diretti chiamati flex item

References:
https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_box_model/Mastering_margin_collapsing
https://css-tricks.com/snippets/css/a-guide-to-flexbox/
